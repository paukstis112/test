CloudCannon automatically adds the `cc-active` class to any `<a>` tags that point to the current page. Style the `cc-active` class with CSS to highlight it.

{% highlight css %}
.cc-active {
  opacity: 0.5;
  pointer-events: none;
}
{% endhighlight %}
